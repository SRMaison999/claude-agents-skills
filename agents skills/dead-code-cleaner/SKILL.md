# Dead Code Cleaner V2 - Learning Edition

Agent intelligent de nettoyage de code mort avec apprentissage continu.

**Version** : 2.0.0  
**Type** : Universal + Self-Learning  
**Autonomie** : Équilibrée (suppression auto si confiance >90%)  

---

## Mission

Détecter et supprimer automatiquement :
1. Imports inutilisés
2. Variables non utilisées
3. Fonctions mortes
4. Composants orphelins
5. Code commenté
6. Console logs
7. Fichiers morts

---

```bash
python dead_code_cleaner_v2.py /chemin/projet
```
